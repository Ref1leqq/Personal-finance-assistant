def create_db(): ¶

Создает базу данных и таблицы, если они еще не существуют.

Таблицы: - users: Хранит данные пользователей. - goals: Хранит финансовые цели пользователей. - reminders: Хранит напоминания пользователей. - transactions: Хранит транзакции (доходы и расходы).
